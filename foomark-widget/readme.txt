Foomark WordPress Widget displays your Foomark bookmarks in a widget.

== Description ==

Foomark WordPress Widget uses Foomark's API to take a user's foomarks in JSON format and displays them in a widget. The two settings fields needed to make it work are a Foomark username and the number of foomarks you want to display.

After Installing the plugin, go to Settings -> Foomarks to enter your Foomark username and the number of foomarks to show in the widget.

== Installation ==

1. Upload the folder `foomark-widget` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= What is Foomark? =

Foomark is a social bookmarking service. You can register for free at http://www.foomark.com/register/
